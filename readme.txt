Update WordPress.com themes seamlessly with the rest of your WordPress updates.

== Description ==

Your WordPress.com themes might need updating, but who wants to update them manually? This plugin enables seamless updating of your WordPress.com themes right in the WordPress updater, no fuss, no muss, even on multisite networks.

== Installation ==

1. Upload wordpresscom-theme-updates to /wp-content/plugins/.
2. Activate plugin through the WordPress Plugins menu.
3. Install updates when there are updates. :)

== Frequently Asked Questions ==

= Where is the plugin's admin page? =
There isn't one! Go to Dashboard -> Updates and theme updates will just present themselves when there is one.
